# Abandoned-Cart-EmailAbandoned Cart Email

Responsive abandoned cart email template built with HTML and CSS for e-commerce customer retention campaigns.

## Features

- Responsive layout
- Personalized content placeholders
- Product showcase section
- Discount code promotion
- Outlook-compatible CTA button (VML)
- Social media footer
- Mobile-friendly design
